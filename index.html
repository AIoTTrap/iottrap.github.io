Introduction

PestDetective is an AI-powered tool designed for pest control and management. This project aims to create a comprehensive platform that utilizes computer vision and machine learning algorithms to identify pests and monitor their movements. The platform includes two key features - Pest Detection, and Teachable Machine - that work together to provide a complete pest control solution.

Backround

Effective pest control is vital for agricultural industries, as pests can cause significant damage to crops, resulting in economic losses and reduced food production. Traditional pest control methods often face challenges such as labor-intensive processes, delays in pest identification, and limited monitoring capabilities. These shortcomings necessitate the development of an advanced solution that harnesses the power of AI, computer vision, and machine learning algorithms to overcome these challenges and revolutionize agricultural pest control.

Key Features

1. Pest Detection

The Pest Detection feature lies at the core of PestDetective, utilizing advanced computer vision algorithms to analyze images or video footage and accurately identify pests. Through extensive training on annotated datasets, deep learning models within the platform can recognize a wide range of pests with high accuracy. Real-time monitoring capabilities and an alert system can notify farmers of pest presence, allowing for immediate action to mitigate potential crop damage.

Pest Photo Upload:

The Pest Photo Upload feature allows users to upload a photo of a pest and receive instant identification of the type of pest with bounding boxes displayed around it. This feature is useful for identifying pests in crops and fields for proper control measures. This feature currently detects 5 classes:
Beetles
Caterpillars
Snails
Ants
Insects

ESP32 Detection:

The ESP32 Detection feature allows users to receive notifications of pest movement detected by an ESP32 device. This feature is particularly useful for monitoring pests in real-time and preventing infestations.

2. Teachable Machine

The Teachable Machine feature empowers farmers to actively participate in training the AI model to identify specific pests relevant to their agricultural settings. Farmers can upload their own images of pests, enabling the system to learn and adapt to new pest species. The user-friendly interface facilitates the training process, allowing farmers to customize the platform according to their specific needs and optimize pest control strategies accordingly.

Note: This system may also be used for any other classes apart from 'Pests'.

Implementation and Technical Details

Obtaining YOLO weights for PEST DETECTION feature

The YOLOv3 model was trained on a custom dataset of insect images using Darknet, an open-source neural network framework. The Google Open Images Dataset v7 was utilized for downloading the image dataset for insects. The image dataset consisted of .jpg files and corresponding .txt files with matching names. Each line of the .txt file was formatted as follows:

<object-class> <x_center> <y_center> <width> <height>

The bounding box annotations were converted to the required YOLO format for utilizing the YOLO detector.

Data Preprocessing for TEACHABLE MACHINE feature

PestDetective utilizes a combination of technologies to deliver its pest control capabilities. The platform integrates computer vision libraries, including OpenCV, and powerful deep learning frameworks such as TensorFlow. This integration enables the platform to efficiently process visual data and extract meaningful information for pest identification.

The data preprocessing stage is a crucial step in preparing the large datasets of pest images for training. It involves cleaning the data and annotating it with relevant labels to facilitate supervised learning. By accurately labeling the images with corresponding pest species, the platform can learn from these annotated datasets during the training phase.
